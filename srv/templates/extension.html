Bookmark Manager Extension

Save bookmarks from any page with one click

Installation

1

Download the extension

Click the button below to download the extension zip file.

Download Extension
2

Extract the zip file

Unzip the downloaded file to a folder on your computer.

3

Open Chrome Extensions

Go to chrome://extensions in your browser.

Enable Developer mode (toggle in top right).

4

Load the extension

Click "Load unpacked" and select the extracted folder.

Done!

Click the extension icon in your toolbar to save any page as a bookmark.

Features

One-Click Save

Save any page with a single click

Auto-Tags

Add tags to organize your bookmarks

Source Detection

Auto-detects Instagram, LinkedIn, etc.

Auto Description

Extracts page description automatically
